pglite-wasi-17.tar.gz is the PGlite 17.x WASI binary, extracted from:

  https://github.com/f0rr0/pglite-oxide

Specifically from the file: pglite-oxide/assets/pglite-wasi.tar.xz

This is the same binary that pglite-oxide (a Rust wrapper around PGlite) ships
with. It contains PostgreSQL 17.x compiled to WASI (WebAssembly System Interface)
by the Electric SQL pglite project (https://github.com/electric-sql/pglite).

The tar.gz was created by extracting the tar.xz and re-compressing as gzip
for use with Go's embed and archive/tar packages.

Contents:
  tmp/pglite/bin/pglite.wasi    - The main PostgreSQL WASI binary (~22MB)
  tmp/pglite/bin/initdb          - Symlink to pglite.wasi
  tmp/pglite/bin/postgres        - Symlink to pglite.wasi
  tmp/pglite/share/postgresql/   - PostgreSQL share files (BKI, SQL, configs)
  tmp/pglite/lib/postgresql/     - PostgreSQL extensions
  tmp/pglite/password            - Default password file
